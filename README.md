
## Instalación
***
```
$ git clone https://github.com/DouglasGuacaran/testbike.git
$ cd ../ruta/al/archivo
$ ejecutar
$ npm install
$ npm python3 manage.py startserver

$ en la primera tarea de este proyecto se realiza 
$ una consulta a la BD API de bikeSantiago, primeramente
$ se realizó llenado de la BD en PostgreSQL con los datos de 
$ la API.
$ Cuando se requiere se solicita actualización de la 
$ disponibilidad de la información de la estación de 
$ bicicletas que se encuentran disponibles en la zona.
$ se presenta en forma de tarjeta la información de cada 
$ estación.

$ para la segunda parte de este test se pide realizar 
$ webscrapping de una página que contiene una tabla
$ se elabora un Script para ello y se inserta en una vista
$ de la página, se lleva la información obtenida a la BD
$ y se presenta en una nueva tabla.
